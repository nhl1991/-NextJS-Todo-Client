# TODOLIST
## 📖 Overview
バックエンド学習のため、「Next.js」と「Nest.js」を用いて作成した簡単なTODOアプリです。

---

## 🛠 Tech Stack
### Framework / Library
- Next.js  
- React  
- Nest.js  

### Language
- TypeScript  
- HTML5  
- CSS3  

---

## ✨ Features
- ログイン、ログアウト  
- TODO作成、修正、削除  
- TODOの公開 / 非公開設定  
- 公開されたTODOページ、ユーザーのTODO一覧ページ
